# Issue #475 実装計画からの逸脱記録

## 1. タイムライン軸範囲の today 拡張を lo/hi 双方に広げた

- **元の計画**: プロトタイプ踏襲。軸上限 `hi` を今日まで拡張（`hi = max(hi, today)`）。
- **実際の実装**: 軸下限 `lo` も今日まで拡張（`lo = min(lo, today)` / `hi = max(hi, today)`）。
- **逸脱の理由**: プロトの `hi` のみ拡張だと、全期間が未来（今日 < 全開始日）のとき今日マーカーが軸左外に出て見えない。受け入れ条件「参照日（今日）のマーカー表示」を全ケースで満たすため、両端を today まで拡張して今日を必ず軸内に収める。単体テスト「今日が全期間より過去（全て将来）でも今日マーカーは軸内に収まる」で固定済み。
