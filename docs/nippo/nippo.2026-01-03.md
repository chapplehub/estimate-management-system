### 10:22 - psコマンドの使い方

psコマンドでプロセススナップショットが見れる
psは自分のターミナルのプロセスだけ
ps auxは全プロセス

```
$ ps
  PID TTY          TIME CMD
 1234 pts/0    00:00:00 bash
 5678 pts/0    00:00:00 ps
```

### 10:24 - systemdについて

systemdはLinuxの全プロセスの親分(pstreeコマンドで一番上に出てくる)

### 10:25 - daemonとは

systemd、sshd、dockerdの末尾のdはdaemonのd (バックグラウンドで常駐して動き続けるプロセス)

### 10:26 - TTYでデーモン判別

psコマンドで表示される TTY が ? = ターミナルに紐づいてない = デーモン！

### 10:29 - デーモンとクライアントの違い

sshd,httpd,mysqldはあくまでリクエストを受け取る常駐デーモンなので自分からリクエストを発信する側(ssh,curl,wget,mysql)ではない。
