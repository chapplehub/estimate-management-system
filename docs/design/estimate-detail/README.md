## プレビューの開き方

最も手軽なのは**自己完結版**をブラウザで開く方法:

```
docs/design/estimate-detail/見積詳細画面 (standalone).html
```

ローダー版（`見積詳細画面.html`）を使う場合は、同階層の `.jsx` / `data.js` を相対パスで読むため、ファイルを**移動・分割しない**こと（後述）。

## 構造を保持している理由

`見積詳細画面.html` と `見積詳細画面-standalone-src.html` は、`<script src="data.js">` / `<script type="text/babel" src="item-row.jsx">` のように**同階層の相対パス**で JSX/JS を読み込む。そのため JSX をサブフォルダへ分けるとローダー版プレビューが壊れる。元 zip のフラット構造をそのまま保持しているのはこのため。

## 取り込み時に除外したもの

- `.thumbnail`（WebP）: デザインツールのカード用サムネイル画像。設計成果物ではないため除外。
